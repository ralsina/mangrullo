Mangrullo

Docker Container Update Automation

A Crystal-based tool that automatically monitors and updates your Docker containers, similar to Watchtower but with a focus on simplicity and reliability.

Installation

From Source

# Install Crystal (>= 1.16.3)
# Clone the repository
git clone https://github.com/ralsina/mangrullo.git
cd mangrullo

# Install dependencies
shards install

# Build the project
shards build

# Run it
./bin/mangrullo --once

Using Docker

# Build the image
docker build -t mangrullo .

# Run once
docker run --rm \
  -v /var/run/docker.sock:/var/run/docker.sock \
  mangrullo --once

Usage

Basic Commands

# Check for updates once
mangrullo --once

# Monitor every 5 minutes
mangrullo --interval=300

# Dry run to see what would be updated
mangrullo --dry-run

# Allow major version upgrades
mangrullo --allow-major

Container Filtering

# Check specific containers
mangrullo --once flatnotes atuin

# Flexible name matching
mangrullo --once /flatnotes atuin /radicale

Architecture

Mangrullo is built with a modular architecture using Crystal:

Core Modules

  • Docker Client - Docker API wrapper and container operations
  • Image Checker - Version checking and update detection
  • Update Manager - Coordinates updates with container filtering
  • Configuration - Command-line argument parsing

Key Features

  • Container recreation (not just restart)
  • Flexible container name matching
  • Multi-registry support
  • Comprehensive error handling
